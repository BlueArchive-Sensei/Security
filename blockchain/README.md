# 区块链Demo项目

## 📋 项目简介

这是一个功能完整的区块链演示项目，实现了区块链的核心功能，包括：

- 🔗 区块创建和链接
- ⛏️ 工作量证明挖矿
- 🔒 防篡改验证机制
- 💰 交易系统
- 📁 数据持久化
- 🎮 交互式演示

## 🚀 快速开始

```bash
# 进入演示目录
cd demo

# 一键启动（推荐）
python3 run_demo.py

# 或直接运行演示
python3 demo.py

# 或运行功能测试
python3 test_blockchain.py
```

## 📁 项目结构

```
blockchain/
├── README.md           # 项目概览（本文件）
└── demo/              # 演示代码目录
    ├── blockchain.py       # 区块链核心实现
    ├── demo.py            # 综合演示脚本

    ├── run_demo.py        # 一键启动脚本
    ├── requirements.txt   # 项目依赖
    ├── README.md         # 详细文档
    └── *.json            # 运行生成的数据文件
```
